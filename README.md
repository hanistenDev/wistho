# Projekt-Screenshots

## Wo die Bilder hin müssen:

Lege deine Screenshots in diesen Ordner: **`public/projects/`**

## Benennung der Dateien:

Für jedes Projekt benötigst du **2 Bilder**:

### Format:
- `<projekt-id>-desktop.jpg` → MacBook-Ansicht
- `<projekt-id>-mobile.jpg` → iPhone-Ansicht

### Beispiele:

**Webwind:**
- `webwind-desktop.jpg`
- `webwind-mobile.jpg`

**Restaurant Lago:**
- `restaurant-lago-desktop.jpg`
- `restaurant-lago-mobile.jpg`

**FitBox:**
- `fitbox-desktop.jpg`
- `fitbox-mobile.jpg`

**Maler Nova:**
- `maler-nova-desktop.jpg`
- `maler-nova-mobile.jpg`

## Empfohlene Bildgrößen:

- **Desktop:** 1440x900px oder größer (16:10 Verhältnis)
- **Mobile:** 375x812px oder größer (iPhone 14/15 Verhältnis)

Die Bilder werden automatisch mit `object-contain` skaliert, sodass sie perfekt in die Frames passen.
